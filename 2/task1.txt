var a = 1, b = 1, c, d;
1) c = ++a; alert(c);           // 2
2) d = b++; alert(d);           // 1
3) c = (2+ ++a); alert(c);      // 5
4) d = (2+ b++); alert(d);      // 4
5) alert(a);                    // 3
6) alert(b);                    // 3

Пункты 1 и 2:
Постфиксный инкремент возвращает изначальное значение и только потом увеличивает значение.
Префиксный инкремент увеличвает значение и возвращает.

Пункт 3:
После инкремента переменной a она имеет значение 2, далее к 2 мы прибаливи a с префиксным инкрементом в итоге 2 + 3 = 5.

Пункт 4:
После инкремента переменной b она имеет значение 2, далее к 2 мы прибаливи b с постфиксным инкрементом в итоге 2 + 2 = 5.

Пункты 5 и 6:
Было произведено по 2 инкрмента переменных a и b в итоге мы и получили 3 в каждой из переменных.
